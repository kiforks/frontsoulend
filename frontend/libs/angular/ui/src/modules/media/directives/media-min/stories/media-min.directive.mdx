# MediaMin

The `MediaMinDirective` uilies a **min-width** media query to control content visibility based on the specified breakpoint.

## Usage

Use the directive in your component template to display content only from a specific breakpoint and larger:

```html
<div *uiMediaMin="'md'; or: value; and: value; else: value">
	This content will be visible from the 'md' breakpoint onward.
</div>

<ng-template #elseRef>
	<p>Fallback content when the conditions are not met.</p>
</ng-template>
```
