# MediaOnly

The `MediaOnlyDirective` uilies a media query to control content visibility based on a **specific breakpoint**. The content will be visible **only** at the specified breakpoint and hidden at all others.

## Usage

Use the directive in your component template to display content **only** at a specific breakpoint:

```html
<div *uiMediaOnly="'md'; or: value; and: value; else: value">
	This content will only be visible at the 'md' breakpoint.
</div>

<ng-template #elseRef>
	<p>Fallback content when the conditions are not met.</p>
</ng-template>
```
