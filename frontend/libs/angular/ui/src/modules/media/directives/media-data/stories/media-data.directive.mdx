# MediaData

The `MediaDataDirective` allows you to **access media-related observables** from the `MediaService` directly in your component templates.
It enables dynamic bindings based on media breakpoints.

## Usage

Use the directive in your component template to expose media breakpoints as local template variables:

```html
<some-component
	*uiMediaData="let mediaMobile = mediaMobile; let mediaDesktop = mediaDesktop"
	[icon]="(mediaMobile | async) ? 'icon-1' : 'icon-2'"
	[isTransparent]="!!(mediaDesktop | async)"
></some-component>
```
