# MediaBetween

The `MediaBetweenDirective` uilies a **range-based media query** to control content visibility between two specified breakpoints

## Usage

Use the directive in your component template to display content **only between two breakpoints**:

```html
<div *uiMediaBetween="['md', 'lg']; or: value; and: value; else: value">
	This content will only be visible for screen sizes between the 'md' and 'lg' breakpoints.
</div>

<ng-template #elseRef>
	<p>Fallback content when the conditions are not met.</p>
</ng-template>
```
