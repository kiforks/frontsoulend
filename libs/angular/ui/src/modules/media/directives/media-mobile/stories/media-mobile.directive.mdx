# MediaMobile

The `MediaMobileDirective` uilies a **mobile-specific media query** to control content visibility.
The content will only be visible when the viewport matches a mobile device breakpoint.

## Usage

Use the directive in your component template to display content **only on mobile devices**:

```html
<div *uiMediaMobile>This content will only be visible on mobile devices.</div>

<ng-template #elseRef>
	<p>Fallback content when the conditions are not met.</p>
</ng-template>
```
